    public class Tile
extends Light
    Represents a LIFX Tile (or Candle - aka, matrix) light

    Technically speaking, Lights (bulbs) are not Tiles, but since they share common API calls, we ignore that definition issue and implement the Light feature set.

        Set the colour zones for the tile
        Parameters:
        tile_index - the starting tile to change
        length - the number of devices in the chain to change starting from the index
        x - the x co-ordinate to start applying colors from
        y - the y co-ordinate to start applying colors from
        width - the width of the square you're applying colors to - this should be 8 for the LIFX Tile and 5 for the LIFX Candle
        duration - the time it will take to transition to new state in milliseconds.
        colors - the colours to apply to each zone
