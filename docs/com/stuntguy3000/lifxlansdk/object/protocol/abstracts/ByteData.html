    public interface ByteData
    Represents an object that has an associated byte data structure and allows conversions to and from byte arrays

    Not all ByteData objects may have both, if any, functions defined - this depends on the usage for the object throughout the application. Some objects do not need to be decoded, and some do not need to be encoded.
